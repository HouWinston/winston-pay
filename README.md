# 🚀 Winston API 极速起步指南 (小白专属)

只需 3 分钟，完成以下三个步骤，即可无缝体验全球最顶尖的 AI 大模型（含 DeepSeek-R1、Claude 3.5、GPT-4o 等）。

---

### 第一步：领钥匙 (生成你的专属密码)
1. 登录控制台 `winstonhou.app`，点击左侧菜单的 **【钱包】** -> **【充值】**，使用兑换码获取额度。
2. 点击左侧菜单的 **【令牌】** -> 点击蓝色按钮 **【+ 添加新的令牌】**。
3. 直接点击 **【提交】**（无需修改任何默认设置）。
4. 在生成的列表中，点击**绿色的复制图标**，复制那串以 `sk-` 开头的超长密码。（请勿泄露给他人）

---

### 第二步：配引擎 (连接专属聊天室)
*这是最关键的一步，我们需要把您的钥匙插进聊天室里。*
1. 点击控制台左侧的 **【聊天】** -> 进入 **【智能对话框】**。
2. 在弹出的黑色聊天界面中，点击 **左下角的 ⚙️ (设置图标)**。
3. 在左侧菜单选择 **【语言模型】** -> 右侧找到并点击 **【OpenAI】**。
4. 此时您会看到两个重要的输入框，请精确填入：
   * **API Key**：粘贴您刚刚在第一步复制的 `sk-...` 密码。
   * **API 代理地址**：手动输入 `https://winstonhou.app/v1` （⚠️ 极其重要：结尾必须带 `/v1`，不能有空格）。
5. 填好后，向下滑动一点，点击 **【连通性检查】** 按钮。如果提示成功，恭喜您，引擎已点火！

---

### 第三步：选模型 (解锁全系顶级 AI)
*系统默认只显示几个基础模型，按以下操作解锁全部 100+ 款旗舰模型：*
1. 依然在刚才的 **【OpenAI】** 设置页面，滑动到最底部。
2. 找到并点击右下角的文字按钮 **【获取模型列表】**。
3. 系统会瞬间为您拉取本站所有的最新旗舰模型（含 DeepSeek、Claude 等）。
4. **大功告成！** 关闭设置界面，在主页顶部的下拉菜单中选择您喜欢的模型，直接开始提
